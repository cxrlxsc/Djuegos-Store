# Djuegos Store

Djuegos Store es una tienda web estática enfocada en productos de anime, gaming, geek y coleccionables. El proyecto presenta un catálogo de productos, una página de carrito y una experiencia visual sencilla para mostrar los artículos disponibles.

## Características

- Página de inicio con presentación de la tienda.
- Catálogo de productos con tarjetas informativas.
- Botones de compra que redirigen a la página de productos.
- Funcionalidad de carrito usando almacenamiento local del navegador.
- Modal de detalles para ver información ampliada de cada producto.
- Diseño responsivo y estilo visual coherente.

## Estructura del proyecto

- index.html: página principal de la tienda.
- productos.html: catálogo de productos.
- carrito.html: vista del carrito de compras.
- contacto.html: página de contacto.
- css/: archivos de estilos.
- img/: imágenes del proyecto.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Cómo abrir el proyecto

Puedes abrir cualquiera de los archivos HTML directamente en tu navegador, por ejemplo:

- index.html
- productos.html
- carrito.html

Si prefieres usar un servidor local, puedes ejecutar desde la carpeta del proyecto:

```bash
python -m http.server 8000
```

Luego abre en tu navegador:

```text
http://localhost:8000/
```

## Funcionalidades destacadas

- Navegación entre páginas.
- Agregar productos al carrito.
- Ver detalles completos de un producto en una ventana modal.
- Mostrar y actualizar el contenido del carrito dinámicamente.
